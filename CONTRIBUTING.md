## How Can I Contribute?

### Reporting Bugs

- Use the GitHub Issues page
- Include detailed steps to reproduce
- Mention your Android version and device

### Suggesting Features

- Open an issue with the "enhancement" label
- Describe the feature and its use case
- Explain why it would be valuable

### Pull Requests

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Test thoroughly
5. Submit a PR with a clear description

### Code Style

- Follow Kotlin coding conventions
- Use meaningful variable/function names
- Add comments for complex logic
- Format code with Android Studio formatter

### Testing

- Test on multiple Android versions
- Verify both portrait and landscape orientations
- Check edge cases (no media, no permissions, etc.)

## Development Setup

1. Install Android Studio Hedgehog or later
2. Clone the repository
3. Open in Android Studio
4. Wait for Gradle sync
5. Run the app
