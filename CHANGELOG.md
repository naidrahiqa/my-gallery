# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-12-29

### Added

- Initial release of My Gallery
- All Media view with grid layout
- Folder organization view
- Favorites functionality
- Trash system with restore capability
- Multiple sorting options:
  - Sort by Title (A-Z)
  - Sort by Date
  - Sort by Duration (for videos)
  - Sort by Size
- Sort order toggle (Newest/Oldest first)
- Material Design 3 UI
- Runtime permission handling for Android 13+
- Efficient image loading with Coil
- Video thumbnail support

### Technical

- MVVM architecture implementation
- Jetpack Compose UI
- Navigation Component
- MediaStore API integration
- Support for Android 7.0+ (API 24+)
